# My-Restaurant-

My Restaurant is a lightweight, interactive web application designed for a restaurant to showcase its menu and facilitate digital ordering. It provides a seamless user experience for customers to browse food categories, manage a virtual cart, and choose their preferred dining or delivery method.

One-Click Ordering: Users can instantly add items to their cart by clicking on the menu cards.
Real-Time Order Management:
Live Cart: An "Order Summary" section dynamically tracks selected items and their quantities.
Auto-Calculations: The total cost is updated in real-time as users add or remove items.
Flexibility: Provides easy options to remove specific items or cancel the entire order.
Fulfillment & Delivery Options:
Eat in Restaurant: A quick "Dine-In" option for local guests.
Home Delivery System: A dedicated delivery module that:
Collects customer details (Name, Phone, Address).
Live Location Integration: Features a "Use Live Location" button that utilizes the browser's Geolocation API to capture precise delivery coordinates.
Responsive Design:
The interface is designed to be user-friendly across different devices, featuring a clean layout with quick navigation (Home, Menu, Order, Contact).
